# Moony's World — design notes

## Palette
- Canvas / cream: #FBF3E3
- Ink (text): #33291f
- Red #E8503A · Mustard #EFA72C · Green #3F8F5A · Blue #3E7CC4 · Pink #EE8AA6

## Type
- Display: Baloo 2 · Body: Nunito · Hand accent: Gochi Hand

## Naming (how we refer to parts of the page)
- Canvas — full page background
- Top bar — sticky strip (title, EN/PT/IT, Sending a gift)
- Hero — banner with moon/star/sun + title
- Note — white "shared list" family message
- Progress bar — gifts-spoken-for tracker
- Section — a colour category group; its flag label = section tag
- Card — one item; picture = thumbnail; green button = claim button
- Gift panel — pink "Sending a gift?" box with address
- Swatches — colour dots that change the canvas
